# Calm New Tab — Landing Page

Landing page for [Calm New Tab](https://calmnewtab.com) — a Chrome extension that replaces the new tab page with live nature video backgrounds, a clock, and weather.

## Pages

- `index.html` — Main landing page with live demo
- `features.html` — Features overview
- `privacy.html` — Privacy policy
- `terms.html` — Terms of service

## Tech

- Vanilla HTML/CSS/JS
- Google Fonts (Inter, Manrope)
- Pexels API for video backgrounds
- Open-Meteo API for weather
- 6 languages (EN, RU, UK, DE, FR, ES)

## Deploy

Hosted on GitHub Pages at [calmnewtab.com](https://calmnewtab.com).
